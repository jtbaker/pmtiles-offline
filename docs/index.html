PMTiles Offline

Enable offline maps in Progressive Web Apps with IndexedDB storage

Get Started View on GitHub Try Demo

Why PMTiles Offline?

True Offline Support

Store entire PMTiles archives in IndexedDB for complete offline functionality. Your maps work even when there's no network connection.

Efficient Storage

PMTiles' cloud-optimized format combined with IndexedDB provides efficient local storage. Only download what you need, when you need it.

Easy Integration

Drop-in replacement for standard PMTiles sources. Works seamlessly with MapLibre GL and other mapping libraries.

Progressive Web Apps

Perfect for PWAs that need offline map capabilities. Enable field data collection, navigation, and location-based services without connectivity.

Reduce Server Load

Cache tiles locally to reduce bandwidth costs and server load. Download once, use forever.

Open Source

MIT licensed and built on open standards. Contribute, customize, or integrate into your projects freely.

Installation

npm install pmtiles-offline

Quick Start

Get up and running in minutes with this simple example:

import { PMTiles } from "pmtiles";
import { IndexedDBSource } from "pmtiles-offline";

// Open or create an IndexedDB database
const dbname = "offline-pmtiles";
const tablename = "offline-pmtiles";
const db = await IndexedDBSource.openDb(dbname, tablename);

// Fetch a PMTiles file from a server (one-time download)
const filename = "my-map.pmtiles";
const response = await fetch("https://example.com/my-map.pmtiles");
const buffer = await response.arrayBuffer();
const blob = new Blob([buffer], { type: "application/octet-stream" });

// Create the offline source and store the PMTiles file
const offlineSource = new IndexedDBSource(db, filename, tablename);
await offlineSource.setSource({ filename, blob });

// Use the offline source with PMTiles
const pmtiles = new PMTiles(offlineSource);
const tile = await pmtiles.getZxy(0, 0, 0);

MapLibre GL Integration

import { PMTiles, Protocol } from "pmtiles";
import { IndexedDBSource } from "pmtiles-offline";
import maplibregl from "maplibre-gl";

// Setup PMTiles protocol
const protocol = new Protocol();
maplibregl.addProtocol("pmtiles", protocol.tile);

// Store PMTiles file in IndexedDB (same as above)
const db = await IndexedDBSource.openDb("offline-pmtiles", "tiles");
const offlineSource = new IndexedDBSource(db, "my-map.pmtiles", "tiles");
// ... download and store file ...

// Register with protocol
const pmtiles = new PMTiles(offlineSource);
protocol.add(pmtiles);

// Create map using offline source
const map = new maplibregl.Map({
  container: "map",
  style: {
    version: 8,
    sources: {
      offline_map: {
        type: "vector",
        url: "pmtiles://my-map.pmtiles"
      }
    },
    layers: [/* your layers */]
  }
});
